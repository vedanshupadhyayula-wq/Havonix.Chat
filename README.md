# Private Chat App

A simple real-time chat application where users can join with just their name and see who's online.

## Features
- Enter name to join
- Real-time messaging
- See online users
- WebSocket-based communication

## Local Setup

```bash
pip install -r requirements.txt
python server.py
```

Visit `http://localhost:8080`

## Deploy to Render (Free)

1. Push to GitHub
2. Go to [render.com](https://render.com)
3. Create new Web Service
4. Connect your GitHub repo
5. Set start command: `python server.py`
6. Deploy!

Your friends can access at: `https://your-app-name.onrender.com`

## Deploy to Replit (Free)

1. Go to [replit.com](https://replit.com)
2. Click "Import from GitHub"
3. Paste your repo URL
4. Click "Run"
5. Share the Replit URL with friends

## Deploy to Railway (Free tier)

1. Go to [railway.app](https://railway.app)
2. Connect GitHub
3. Select your repo
4. Deploy!
